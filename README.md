# pomdor
A Team shared timer ispired by the pomodoro tecnique
## How to build
simply clone the repo
```sh
git clone git@github.com:evoluzione/pomdor.git 
cd pomdor
npm install
```
edit the timer.json inside src
```sh
cd src
vim timers.json
```
and then build your shared timer
```sh
npm run make
```
